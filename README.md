# JAVA_Project_25040201
Factory Inventory &amp; Tool Management System

## An inventory management application for factories/workshops that tracks equipment availability and highlights low stock levels to prevent component shortages.

* Add / remove / update items
* Create items with names and base quantity
* Low stock alert when the app is launched -  low stock when quantity <=10 
* Click the refresh button to synchronize both parts of the application
* Everything is connected to an SQL database: all information about the items is stored there.
